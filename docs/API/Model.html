ALLOW_INCOMPLETE

This symbol can be used to create models with missing "required" fields. Note that this value is inherited, and so can be referenced from your own model's name rather than using Model.ALLOW_INCOMPLETE. E.g. 

  class Something extends Model {
    // ...
  }

  Something.create(undefined, Something.ALLOW_INCOMPLETE);
