DummyStore

This is a dummy store that gets used as replacement for filesystem-store.js in browser context by all bundlers that respect the package.json "browsers" property when set to a mapping object rather than a single entry point string. See https://docs.npmjs.com/cli/v7/configuring-npm/package-json#browser
